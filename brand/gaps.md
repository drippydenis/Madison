## Project Proposal — Top Gap (150–200 words)

**Proposed Madison Build: Data Pipeline Portfolio Generator**

The top gap blocking my aspiration is the absence of a public, runnable data pipeline that a hiring manager at a private tech company can actually inspect. My FCEDA work proves I can build pipelines, but it lives nowhere a stranger can find it. My Navy Federal work is confidential. The gap is real and the fix is specific.

The proposed Madison component is a "Data Pipeline Portfolio Generator" recipe. The entire artifact commits to GitHub with a one-command run instruction.

Success condition: by July 15, 2026, the repo is live, the README explains the pipeline in plain English, and the output visualization is embedded. A hiring manager can clone it in under two minutes. That is the difference between claiming pipeline experience and proving it.
